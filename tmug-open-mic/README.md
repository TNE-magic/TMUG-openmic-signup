# The Magic Underground — Open Mic Manager

A web app for managing TMUG open mic nights at Sisyphus Brewing.

**2nd Tuesday of every month • 7:00–9:00 PM**

## Features

- **Public Sign-Up** — Performers submit their name, effect, time needed, and more
- **Anonymous Feedback** — Performers and audience can leave anonymous feedback
- **Admin Dashboard** (password protected) — Manage sign-ups, run sheets, effect history, performer CRM, feedback, and rules reference
- **Effect History** — Track what everyone's performed to catch repeats
- **Run Sheet Builder** — Set the show order with auto-calculated timing
- **Performer CRM** — Contact log with tags, notes, and show counts

## Tech

Built with React + Vite. Data stored in browser localStorage.

## Setup

```bash
npm install
npm run dev
```
